# Marshmallow - Phaser3 game using Typescript and webpack

- Typescript support.
- Webpack

## Usage

### Install Typscript

```
npm install typescript -g
```

### Start

```
$ git clone https://github.com/davjo664/marshmallow
$ cd marshmallow
$ npm install
$ npm run dev
```

### Build

```
$ npm run build
```
